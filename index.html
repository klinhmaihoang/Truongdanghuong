🔥 Thiết lập Firebase — Sync đa thiết bị
App cần kết nối Firebase Firestore để sync dữ liệu realtime giữa 10+ thiết bị.
Làm 1 lần duy nhất, tất cả mọi người dùng chung 1 file HTML này.
Bước 1 — Vào console.firebase.google.com → Tạo project mới (miễn phí)
Bước 2 — Vào Build → Firestore Database → Create database → chọn Start in test mode
Bước 3 — Vào Project Settings (bánh răng) → Your apps → Add app → Web (</>) → Copy firebaseConfig
Bước 4 — Dán vào form dưới đây → Bấm Kết nối
Lưu ý — Sau khi kết nối, gửi file HTML này cho cả team. Ai mở cũng dùng chung 1 database, dữ liệu sync realtime.
